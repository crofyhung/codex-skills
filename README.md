# Codex Skills

Personal Codex skills that can be installed into `~/.codex/skills`.

## Install

Use Codex's `skill-installer` with this repository path:

```text
repo: <your-github-username>/codex-skills
path: skills/computer-use-session-hygiene
```

Available paths:

```text
repo: crofyhung/codex-skills
path: skills/browser-task-cleanup
```

After installation, restart Codex so the new skill is discovered.
